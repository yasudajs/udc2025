# 用語集（UDC2025）

最終更新日: 2026-06-14

本ドキュメントは、利用者およびフォークして開発する方向けに、本アプリで使用する主要用語をまとめたものです。

## 1. カテゴリ名（正式）

- AED（apiname: `aed`）
- 医療機関（apiname: `hospital`）
- 公衆無線LAN（apiname: `public_wireless_lan`）
- 指定緊急避難場所（apiname: `evacuation_space`）
- 公衆トイレ（apiname: `public_toilet`）

補足:
- 仕様書内でカテゴリを列挙する場合は、原則として上記の正式名を使用する。
- 実装上の内部カテゴリキー（`freewifi`, `evacuation`, `toilet` など）は、コード説明時にのみ併記する。

## 2. API呼称（正式）

- 正式呼称: BODIK API
- WAPIエンドポイントを明示する場合: BODIK API（WAPI）
- ベースURL: `https://wapi.bodik.jp`

## 3. 実行形態（本プロジェクト）

- サーバー実行形態の表記は `CGI/LSAPI` を使用する。
- 仕様書では WSGI 系の記述は使用しない。

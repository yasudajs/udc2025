# UDC2025

FlaskベースのWebアプリケーション

## セットアップ

### 前提条件
- Python 3.8以上
- pip

### インストール手順

1. リポジトリのクローン
```bash
git clone <repository-url>
cd udc2025
```

2. 仮想環境の作成と有効化
```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Mac/Linux
source venv/bin/activate
```

3. 依存パッケージのインストール
```bash
pip install -r requirements.txt
```

4. アプリケーションの起動
```bash
python app.py
```

## 開発

開発サーバーは http://localhost:5000 で起動します。
